# Cadencia de Optimización — Meta Ads España

## Primeros 7 Días: SOLO Monitorizar

### Día 1-2: Verificación técnica

| Check | Acción |
|---|---|
| ¿Pixel dispara? | Verificar en Events Manager > Test Events |
| ¿Anuncios aprobados? | Revisar status cada anuncio (Active, not Disapproved/In Review) |
| ¿Se gasta presupuesto? | Budget delivery check — si no gasta, ver troubleshooting |
| ¿CAPI activo? | Events Manager > Overview > buscar eventos "Server" |
| ¿CPMs en rango? | €12-40 normal para lead gen España. >€100 = investigar |

### Día 3-7: Observación + negativas únicas

| Check | Acción |
|---|---|
| Entrega por anuncio | ¿Se reparte el gasto? ¿Algún ad acapara todo? |
| CPMs por placement | ¿Algo anómalo? |
| Primeros leads entrando | ¿Calidad aceptable? (verificar en CRM, no solo en Meta) |

**ÚNICA optimización permitida semana 1:** pausar anuncios rechazados por Meta.

**NO TOCAR** presupuestos, audiencias, estructura, creativos.

## Cadencia Post-Lanzamiento

### Diaria (5-10 min)

| Acción | Buscar |
|---|---|
| Check anomalías | ¿Campañas activas? ¿Desaprobaciones nuevas? ¿Gasto normal? |
| Spend pacing | ¿Estamos en línea con presupuesto mensual? |
| Alertas reglas automatizadas | ¿Alguna regla disparó? ¿Algo pausado? |

**NO optimizar diariamente.** Solo detectar emergencias (ads rechazados, tracking roto, gasto desbocado).

### Semanal (30-60 min)

| Acción | Detalle |
|---|---|
| Frecuencia por anuncio | >3 prospecting → creativos nuevos necesarios |
| Pacing check | Gasto vs presupuesto mensual. Ajustar si necesario |
| Métricas por placement | Breakdown > Placement. ¿AN arrastrando calidad? |
| Calidad leads (CRM) | ¿Leads convierten a llamadas/ventas? CPL real vs CPL Meta |
| Creativos rendimiento | ¿Qué hooks funcionan? ¿Cuáles pausar? |
| Comentarios anuncios | ¿Negativos acumulándose? Limpiar |

### Mensual (2-4 horas)

| Acción | Detalle |
|---|---|
| Creative refresh | Nuevos conceptos, hooks, formatos basados en datos mes anterior |
| Reasignación presupuesto | Mover gasto a campañas/ad sets ganadores |
| Auditoría audiencias | ¿Seeds LAL actualizados? ¿Custom audiences frescas? |
| ENCAC review | ENCAC vs CAC Meta. ¿Divergencia? ¿Tendencia? |
| Funnel post-clic | Show rates, speed to lead, tasa cierre. ¿Cuello botella? |
| Report cliente | Resultados, insights, recomendaciones próximo mes |

### Trimestral (1 día)

| Acción | Detalle |
|---|---|
| Auditoría completa cuenta | Full audit (ver audit-checklist.md) |
| Revisión estructura | ¿Consolidar? ¿Expandir? ¿Cambio de fase? |
| Refresh estrategia oferta | ¿Oferta sigue siendo competitiva? ¿Nuevo ángulo? |
| Estacionalidad próximo trimestre | Ajustar presupuesto y creativos para próximo período |
| Competitive intelligence | Ad Library: ¿qué hacen competidores? ¿Nuevos players? |
| Revisión landing pages | ¿Velocidad? ¿CVR? ¿Mobile optimizado? |

## Cuándo Preocuparse

| Señal | Significado | Acción |
|---|---|---|
| 1 día malo | Ruido normal | No tocar |
| 3 días consecutivos malos | Algo cambió | Investigar: tracking, rechazos, problemas entrega |
| 1-2 semanas de caída | Tendencia real | Optimizar con datos reales |
| CPM subió >30% | Competencia, saturación, o problema cuenta | Verificar: estacionalidad, frecuencia, account quality |
| CPL duplicó | Algo está roto | Diagnóstico: tracking → entrega → audiencia → creativo → oferta |
| ENCAC sube pero Meta-CPA estable | Meta sobreatribuye | Investigar view-through, canales canibalizando |

## Jerarquía de Optimización (Meta Ads)

En Meta, el orden de impacto es DIFERENTE a Google Ads:

| Prioridad | Palanca | Impacto | Acción |
|---|---|---|---|
| 1 | **Creativos** | MAYOR | Nuevos conceptos, hooks, formatos. Creative testing continuo |
| 2 | **Oferta / LP** | Alto | Mejorar propuesta, formulario, página destino, CRO |
| 3 | **Presupuesto** | Medio | Redistribuir a ganadores. Incrementos 20% |
| 4 | **Audiencias** | Medio-bajo | Ajustar targeting, exclusiones, seeds LAL |
| 5 | **Pujas** | Bajo | Último recurso. Solo con volumen datos suficiente |

**En Meta, el creativo ES el targeting.** Un creativo nuevo puede cambiar completamente el rendimiento de una campaña sin tocar nada más.

## Regla Anti-Overoptimización

### Ediciones que resetean Learning Phase
- Cambios presupuesto >20%
- Cambios audiencia significativos
- Cambios creative sustanciales
- Cambio optimization event
- Pausa/reactivación campaña

### Ediciones que NO resetean
- Cambio nombre campaña/ad set
- Cambio bid amount (menor)
- Añadir nuevos anuncios (no eliminar existentes)
- Ajustes menores de presupuesto (<20%)

### Principio
Cuantas menos ediciones, más consistente el rendimiento. "No tocar" es frecuentemente la mejor optimización.

## Estacionalidad Española — Ajustes Cadencia

| Período | Ajuste cadencia |
|---|---|
| Agosto (vacaciones) | Reducir presupuesto 30-50%. Pausa tests nuevos. Mantener solo evergreen |
| Pre-Black Friday (oct) | Aumentar producción creativa. Preparar campañas |
| Black Friday-Navidad (nov-dic) | CPMs suben. Aumentar presupuesto ganadores. No testear cosas nuevas |
| Reyes (6 ene) | Último push regalo. Ajustar después |
| Paga extra (jun, dic) | Push presupuesto — mayor poder adquisitivo |
| 3ª semana mes | Reducir ligeramente — menor poder adquisitivo pre-nómina |
